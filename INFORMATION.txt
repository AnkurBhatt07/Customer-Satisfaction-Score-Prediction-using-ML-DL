The output variable is having 5 classes ( multi class classification problem). But one more thing to keep in mind is that the 5 classes in the target/output variable have a relation i.e. they have a order. The five classes are respectively 1,2,3,4,5 and they depict the customer satisfaction scores. Due to this though we can use categorical cross entropy loss function or sparse categorical cross entropy loss function for the problem but that makes us ignore the order in the 5 classes of target. This means that this loss function treats each of the class independent of the other classes . For example it treat the distance between 1 and 2 to be equidistant to the distance between 1 and 5. So it completely ignores the relation between the output classes. To take in cognizance the order of the output classes , we have to use some special ordinal loss function.

Or we can convert this multi class classification problem into a binary problem (promoters and detractors) .
